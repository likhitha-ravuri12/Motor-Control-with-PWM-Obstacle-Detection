# ⚙️ Motor Control with PWM & Obstacle Detection

This project uses Arduino to control motor speed using PWM and automatically stop the motor when an obstacle is detected using an ultrasonic sensor.

## 🔧 Components

* Arduino Uno
* Ultrasonic Sensor (HC-SR04)
* Motor Driver
* DC Motor

## ⚙️ Working

The ultrasonic sensor measures the distance to nearby objects. If an obstacle is detected within a set distance, the Arduino stops the motor. Otherwise, the motor runs with speed controlled using PWM.
